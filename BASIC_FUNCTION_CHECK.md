# 基础功能检查清单

## 目标
**只修复问题，不添加新功能**

---

## 核心功能检查

### 1. 游戏能否启动
- [ ] 打开页面是否正常显示
- [ ] 控制台是否有错误
- [ ] Canvas 是否正确渲染

### 2. 玩家控制
- [ ] 鼠标移动是否正常
- [ ] 光标是否跟随鼠标
- [ ] 暂停/重启按键是否工作

### 3. 载具系统
- [ ] 载具是否显示
- [ ] 载具是否自动前进
- [ ] 网格是否正确渲染

### 4. 武器系统
- [ ] 炮塔是否自动射击
- [ ] 子弹是否正常发射
- [ ] 子弹是否命中敌人

### 5. 敌人系统
- [ ] 敌人是否生成
- [ ] 敌人是否移动
- [ ] 敌人是否攻击载具
- [ ] 敌人死亡是否正确处理

### 6. 资源系统
- [ ] 资源节点是否生成
- [ ] 资源是否能采集
- [ ] 资源数值是否正确更新
- [ ] 弹药是否正确消耗

### 7. 碰撞系统
- [ ] 子弹是否击中敌人
- [ ] 敌人是否攻击到载具
- [ ] 伤害计算是否正确

### 8. 障碍物系统
- [ ] 障碍物是否生成
- [ ] 障碍物是否能挖掘
- [ ] 挖掘完成是否给奖励
- [ ] 载具是否被障碍物阻挡

### 9. UI 显示
- [ ] 资源显示是否正确
- [ ] 波次信息是否正确
- [ ] 血量显示是否正确

### 10. 游戏结束
- [ ] 核心被摧毁是否游戏结束
- [ ] 是否能重启游戏

---

## 检查方法

1. 启动游戏观察
2. 检查控制台错误
3. 逐个测试功能
4. 记录所有问题
5. 只修复，不扩展
