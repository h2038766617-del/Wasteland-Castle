⚠️  重要：不能直接双击 index.html！
   会出现 CORS 错误，必须通过 HTTP 服务器运行。

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【方式 1】使用 VS Code Live Server（最简单，强烈推荐！）

  1. 下载并安装 VS Code
     https://code.visualstudio.com/

  2. 打开 VS Code，点击左侧"扩展"图标（或按 Ctrl+Shift+X）

  3. 搜索 "Live Server"，安装第一个结果
     作者：Ritwick Dey

  4. 在 VS Code 中打开本项目文件夹

  5. 右键点击 index.html
     选择 "Open with Live Server"

  6. 浏览器会自动打开游戏！

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【方式 2】安装 Python 后使用批处理脚本

  1. 下载并安装 Python
     https://www.python.org/downloads/

     ⚠️ 安装时务必勾选 "Add Python to PATH"

  2. 安装完成后，双击 START_SERVER_SIMPLE.bat

  3. 在浏览器访问：http://localhost:8000

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【方式 3】手动在命令行运行（如果已安装 Python）

  1. 按 Win+R，输入 cmd，按回车

  2. 输入以下命令进入项目目录：
     cd /d "完整路径\Wasteland-Castle-claude-cursor-commander-game-Wwqjq"

  3. 输入以下命令启动服务器：
     python -m http.server 8000

  4. 在浏览器访问：http://localhost:8000

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【方式 4】使用 Node.js（如果已安装）

  1. 在命令行进入项目目录

  2. 运行：npx http-server -p 8000

  3. 在浏览器访问显示的地址

【方式 5】使用其他 HTTP 服务器工具

  浏览器扩展：
  - Web Server for Chrome
  - 200 OK (Chrome 扩展)

  软件工具：
  - XAMPP
  - WAMP
  - Nginx

✅ 成功标志：

  浏览器打开后，你应该看到：
  - 黑色背景的游戏画面
  - 标题："光标指挥官 - 敌人攻击测试"
  - 屏幕中间有一个 4×4 的网格
  - 网格中有不同颜色的方块（组件）
  - 鼠标光标变成青色圆点
  - 敌人从边缘生成并移动

  游戏操作：
  - 空格键：暂停/继续
  - D 键：显示调试信息
  - 鼠标：控制无人机光标

❓ 如果还是不行？

  1. 确保使用的是现代浏览器
     Chrome 87+, Firefox 78+, Edge 88+

  2. 打开浏览器控制台（按 F12）
     查看是否有错误信息

  3. 检查 8000 端口是否被占用
     可以尝试其他端口，如 8080, 3000
